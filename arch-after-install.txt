Tweaks:
===GRUB UPDATES===
sudo nano /etc/default/grub 
---remove quiet (if there) 
    GRUB_CMDLINE_LINUX_DEFAULT="quiet...."
    # Uncomment to disable graphical terminal
    GRUB_TERMINAL_OUTPUT=console
--change grub timeout to zero
   sudo nano /etc/default/grub 
   GRUB_TIMEOUT=0
   
   sudo update-grub

===Create a new mirror list, more speed====
sudo pacman -Syy
sudo pacman -S reflector 
OR
sudo pacman-mirror -f 5

===Update System===
sudo pacman -Syyu  

===Add keyboard shortcuts===
--win-e - Gnome, Keyboard shortcuts, Home Folder 
--ctrl-alt-t - Add In ----  gnome-terminal --maximize

===Install Screensavers===
sudo pacman -S xscreensaver
gedit ~/.config/autostart/screensaver.desktop

[Desktop Entry]
Name=Screensaver
Type=Application
Exec=xscreensaver -nosplash

----if it does not work on reboot, copy file to
---->  /etc/xdg/autostart/screensaver.desktop

===Install MS Fonts===
Open pamac and search ttf-ms-fonts 

===Enable TRIM for SSD hard drives===
sudo systemctl enable fstrim.timer (restart to take effect)


===Remove orphan (unused) packages===
In pamac, under Orphans. If some there, remove.
